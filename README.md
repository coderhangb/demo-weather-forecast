## About
This is a simple **weather forecast website** built to practice **API calling** and displaying data on a web page.

## Features
- Search weather by city name.  
- Show temperature, humidity, wind, and weather conditions.  
- Display **5-day weather forecast** with daily summaries.  
- Clean and simple interface.

## Tech Stack
- HTML / CSS / JavaScript  
- Weather API (e.g. OpenWeather)
